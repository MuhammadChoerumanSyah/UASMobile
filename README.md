# UASMobile

# Flutter API
Ini adalah proyek UAS Pemrograman Mobile yang menggabungkan penggunaan API dengan Flutter untuk membuat aplikasi mobile yang menakjubkan. Dalam proyek ini, Anda akan menggunakan kehebatan Flutter untuk mengonsumsi dan menampilkan data dari API yang tersedia dengan elegan dan interaktif. Dengan kemampuan Flutter yang luar biasa, aplikasi mobile yang Anda buat akan menjadi perpaduan sempurna antara fungsionalitas dan desain yang menawan. Bersiaplah untuk menciptakan pengalaman pengguna yang luar biasa dengan proyek ini!
## Instalasi
1. Pastikan Anda telah menginstal Flutter SDK. Jika belum, Anda dapat mengunjungi flutter.dev untuk petunjuk instalasi.
2. Clone repositori ini ke dalam direktori lokal Anda:
   ```shell
   git clone https://github.com/MuhammadChoerumanSyah/UASMobile.git
   ```
4. Masuk ke direktori proyek:
   ```shell
   cd repo-api-flutter
   ```
6. Jalankan perintah berikut untuk menginstal dependensi yang diperlukan:
   ```shell
   flutter pub get
   ```
8. API yang saya gunakan adalah `https://www.themoviedb.org/movie`
